Import Library EBImage

Jika library EBImage dari package BiocManager sudah terinstall maka untuk pertama kita load dahulu librarynya.
Misalkan terjadi eror dalam pemasangan librarynya, kita bisa menggunakan alternatif library jpeg. 

library(EBImage)
library(jpeg)

Read, Disp & Write Image

Reading Image

Beberapa fungsi dasar dari library EBImage adalah untuk membaca, menampilkan dan menuliskan gambar. Untuk membaca gambar kita menggunakan perintah readImage. Langsung saja kita coba.

Bahasa R bisa menampilkan beberapa jenis format gambar yaitu jpeg, png, dan tiff. 

# set lokasi kerja
setwd("C:/Users/glnrmdan/Documents/R/Data_Science_1")
library(EBImage)
# Reading Image
img <- readImage('foto1.JPG')
display(img)

Displaying Image

Lalu kita tampilkan gambarnya menggunakan fungsi display. 

display(img, method="browser")

Tentang Warna Gambar

Ada perbedaan dari gambar bertipe gray-scale berdasarkan properti colorMode dan angka dari dimensinya yaitu 3 (B&W) versus 4(color). colorMode ini bisa digunakan untuk setting warna dari gambar yang akan diolah. Jika kita rubah colorMode menjadi gray-scale maka otomatis dimensi ketiga dan semua dimensi yang lebih tinggi dari array akan dianggap terpisah sesuai dengan frames dari gambar. Disisi lain jika colorMode ini diset menjadi Color maka tiga dimensi tadi akan diasumsikan untuk menangani berbegai perbedaan warna yang ada, serta hanya dimensi keempat dan yang lebih tinggi saja yang digunakan sebagai multiple image frames. Selanjutnya imgcol memiliki 3 channel warna, yang terdiri dari RGB (Red, Green, Blue).

Daerah dari “frames.total” dan "frames.render ditampilkan berdasarkan total dari jumlah frame yang ada didalam gambar, dan jumlah dari frame yang ter-render. Jumlah angka ini dapat diakses dengan menggunakan fungsi numberOfFrames yang dirincikan dengan type didalamnya. 

numberOfFrames(imgcol, type = "render")

Manajemen Warna

Seperti yang sudah dijelaskan sebelumnya jika class dari image ini extends atau berakar dari class array dan menggunakan fungsi colorMode untuk mentimpan tentang informasi mengenai warna dari banyak dimensi data yang harus diatur.

Fungsi colorMode dapat digunakan untuk mengakses, mengubah dan merender mode dari sebuah gambar. Misalnya saja, jika kita mengambil sebuah gambar yang berwarna dan ingin merubahnya kedalam mode gray-scale, maka warna dari gambar ini akan terender berubah menjadi warna gray-scale dengan memisahkan RGB yang ada. Sebenarnya fungsi dari colorMode ini tidak merubah data asli dari gambar, melainkan hanya merubah hasil render gambar dengan package EBImage yang ada di R. 

colorMode(imgcol) = Grayscale
display(imgcol, all=TRUE)

Konversi warna antara gray-scale dan Color dapat dilakukan dengan fungsi channel. Tidak seperti colorMode, channel ini akan merubah nilai intensitas pixel dari gambar yang diolah.

Ada juga fungsi asred, asgreen, dan asblue yang dapat digunakan untuk mengkonversi gambar gray-sclae menjadi gambar yang warnanya lebih spesifik ke hue.

Fungsi dari image juga dapat digunakan untuk mengkonstruksi sebuah warna dari gambar berdasarkan karakter dari vektor atau array yang bernama R colors dituliskan dengan colors() atau bisa juga dengan menggunakan string hexadesimal seperti “#rrggbb” or “#rrggbbaa”.

Brightness, Contrast, dan Gamma

Kita juga dapat merubah kecerahan gambarnya dengan menambahkan angka, mengatur kontras dengan mengalikan, dan mengatur gamma dari gambar dengan melakulan pemangkatan pada data gambar.

Filtering

Filter Linear

Fungsi umum dari Filtering ini adalah untuk membersihkan gambar atau membuat teksture dari gambar menjadi lebih lembut dari aslinya. Didalam package (“EBImage”) yang ada di R kita bisa mengimplementasikan fungsi yang bernama filter2, dan juga fungsi tambahan seperti makeBrush. Walaupun sebenarnya proses Filtering ini dapat dilakukan secara matematis dengan operasi tertentu, namun salah satu cara yang paling efisien adalah menggunakan kode yang sudah ada didalam package (“EBImage”). 

w = makeBrush(size=31, shape='gaussian', sigma=5)
plot(w[nrow(w)+1/2, ], ylab="2", xlab="", cex=0.7)

img_flo = filter2(img, w)
display(img_flo)
## Only the first frame of the image stack is displayed.
## To display all frames use 'all = TRUE'.

Disini kita menggunakan filter Gaussian dengan tingkat filternya berjumlah 5 yang diberikan oleh fungsi sigma. Contoh Filter lainnya seperti “box”, “disc”, “diamond”, dan “line”.

Jika kita memfilter banyak frame maka Filter akan otomatis memproses setiap frame yang dioalah. Untuk lebih mudahnya kita dapat menggunakan fungsi yang bernama gblur untuk melakukan Gaussian Smoothing yang akan disesuaikan otomatis ukuran filternya dengan sigma. 

nuc_gblur = gblur(nuc, sigma=5)
display(nuc_gblur, all=TRUE)

Didalam Signal Processing operasi untuk melakukan penghalusan terhadap gambar lebih menuju kearah filter low-pass. Sebaliknya filter high-pass bertujuan untuk mempertajam gambar yang diolah.

Median Filter

Cara lain yang dapat kita lakukan untuk mengurangi noise pada gambar adalah dengan menerapkan Median Filter. Filter ini dapat melakukan analisa pada gambar pixel demi pixel, sehingga pemroresan gambarnya akan menjadi lebih detil untuk menghilangkan noise. Teknik Filter ini dapat dilakukan dengan menggunakan fungsi medianFilter yang ada didalam package ("EBImage). 

l = length(img)
n = l/10

Operasi Morfologikal

Binary Images merupakan gambar yang memiliki dua jenis pixel dengan nilai 0 dan 1. Dua nilai ini merepresentasikan background dan foreground dari setiap pixel yang ada. Ada dua jenis struktur element dalam penerapannya yaitu erosion dan dilation. 

shapes = readImage(system.file('images', 'shapes.png', package = 'EBImage'))
logo = shapes[110:512,1:130]
display(logo)

kern = makeBrush(5, shape='diamond')
display(kern, interpolate=FALSE)

logo_erode = erode(logo, kern)
logo_dilate = dilate(logo, kern)

display(combine(logo_erode, logo_dilate), all=TRUE)

Tresholding

Global Tresholding

Sebelumnya, dibagian Manipulasi Gambar kita telah mempraktekkan tentang bagaimana caranya megnatur global treshold disebuah gambar. Sekarang kita akan menggunakan sebuah arbitrary cutoff value. Didalamnya kita juga akan menggunakan sebuah method yang bernama Otsu. Method ini digunakan untuk menamiplkan treshold gambar dengan otomatis.

Otsu treshold bisa dilakukan dengan menggunakan fungsi otsu. Ketika kita panggil fungsi tersebut pada frame gambar yang banyak, maka dia akan secara otomatis mengolah setiap frame dari gambarnya. 

treshold = otsu(nuc)
treshold
## [1] 0.6113281 0.5371094 0.4824219
nuc_th = combine(mapply(function(frame, th)frame > th, getFrames(nuc), treshold, SIMPLIFY=FALSE))
display(nuc_th, all=TRUE)

Adaptive Tresholding

Tujuan dari melakukan Adaptive Tresholding adalah untuk membandingkan antara tresholding versisebelumnya dan tresholding yang diperbolehkan di berbagai wilayah gambar. Adaptive Tresholding bekerja dengan membandingkan intensitas setiap pixel baclground terhadap lingkungan lokal dari gambar. Hal ini bisa didapatkan dengan membandingkan gambar asli dengan versi yang telah dihaluskan. 

disc = makeBrush(31, "disc")
disc = disc/sum(disc)
offset = 0.05
nuc_bg = filter2(nuc, disc)
nuc_th = nuc > nuc_bg + offset
display(nuc_th, all=TRUE)

Teknik ini mengasumsikan jika objek (warna putih) relatif jarang tersebar didalam gambar dan lebih didominasi oleh background (warna hitam). Fungsi Adaptive Tresholding menggunakan filter linear dengan box persegi panjang yang disediakan oleh tresh, yang jika diterakan implementasinya akan menjadi lebih cepat jika dibandingkan dengan langsung menggunakan filter2. 

display(thresh(nuc, w=15, h=15, offset=0.05), all=TRUE)

Transformasi Fourier

Fungsi FFT Shift

# FFT SHIFT
#' Fungsi ini berguna untuk melakukan visualisasi Transformasi Fourier dengan frekuensi kosong 
#' komponen ada ditengah spektrum
#' 
#' @param #img_ff adalah Transformasi Fourier dari sinyal 1D, gambar 2D, atau volume 3D.
#' @param dim merupakan jumlah dari dimensi (-1, 1, 2, 3).
#' @return A properly shifted FT of the array.
#' 
fftshift <- function(img_ff, dim = -1) {

  rows <- dim(img_ff)[1]    
  cols <- dim(img_ff)[2]
  # planes <- dim(img_ff)[3]

  swap_up_down <- function(img_ff) {
    rows_half <- ceiling(rows/2)
    return(rbind(img_ff[((rows_half+1):rows), (1:cols)], img_ff[(1:rows_half), (1:cols)]))
  }

  swap_left_right <- function(img_ff) {
    cols_half <- ceiling(cols/2)
    return(cbind(img_ff[1:rows, ((cols_half+1):cols)], img_ff[1:rows, 1:cols_half]))
  }
  
  #swap_side2side <- function(img_ff) {
  #  planes_half <- ceiling(planes/2)
  #  return(cbind(img_ff[1:rows, 1:cols, ((planes_half+1):planes)], img_ff[1:rows, 1:cols, 1:planes_half]))
  #}

  if (dim == -1) {
    img_ff <- swap_up_down(img_ff)
    return(swap_left_right(img_ff))
  }
  else if (dim == 1) {
    return(swap_up_down(img_ff))
  }
  else if (dim == 2) {
    return(swap_left_right(img_ff))
  }
  else if (dim == 3) {
    # Use the `abind` package to bind along any dimension a pair of multi-dimensional arrays
    # install.packages("abind")
    library(abind)
    
    planes <- dim(img_ff)[3]
    rows_half <- ceiling(rows/2)
    cols_half <- ceiling(cols/2)
    planes_half <- ceiling(planes/2)
    
    img_ff <- abind(img_ff[((rows_half+1):rows), (1:cols), (1:planes)], 
                    img_ff[(1:rows_half), (1:cols), (1:planes)], along=1)
    img_ff <- abind(img_ff[1:rows, ((cols_half+1):cols), (1:planes)], 
                    img_ff[1:rows, 1:cols_half, (1:planes)], along=2)
    img_ff <- abind(img_ff[1:rows, 1:cols, ((planes_half+1):planes)], 
                    img_ff[1:rows, 1:cols, 1:planes_half], along=3)
    return(img_ff)
  }
  else {
    stop("Invalid dimension parameter")
  }
}

Segmentasi Gambar

Segmentasi Gambar dapat menampilkan partisi atau komponen dari sebuah gambar, dan secara lebih spresifik berfungsi untuk mengidendtifikasi objek yang ada didalam gambar tersebut. Ada tiga fungsi yang bisa digunakan untuk melakukan Segmentasi Gambar yaitu bwlabel, watersheed, dan propagate. Disini kita akan mencoba gunakan bwlabel. Fungsi ini berguna untuk menemukan setiap pixel yang terhubung dengan background gambar, dan memberi label berupa format integer yang bertambah. 

logo_label = bwlabel(logo)
table(logo_label)
## logo_label
##     0     1     2     3     4     5     6     7 
## 42217  1375  2012   934  1957  1135  1697  1063

Nilai pixel dari gambar logo_label berkisar dari 0 sesuai dengan background hinga jumlah objek yang ada didalamnya. 

max(logo_label)
## [1] 7

Untuk menampirlkan gambarnya kita menormalakan jarak nilainya menjadi (0,1) menggunakan fungsi display. Maka hasil dari pengolahan gambarnya pasti akan memunculkan perbedaan bayangan dari warna abu-abu. 

display(normalize(logo_label))

Ada juga cara lain agar objek yang ada didalam gambar menjadi memilikiwarna secara acak. Caranya adalah dengan menggunakan fungsi yang bernama colorLabels. 

display(colorLabels(logo_label))

Watershed

Beberapa inti yang ada didalam nuc cukup dekat antara satu dengan lainnya dan bisa digabungkan dalam satu objek besar saat ditreshold, seperti yang terlihat di nuc_th. bwlabel akan salah mengidentifikasi mereka sebagai satu kesatuan objek. Alogoritma Watershed akan mengizinkan masalah ini, dia akan mencoba untuk memperlakukan gambar gray-scale seperti relif topografi atau height-map. Peta jarak yang berisi jarak antara pixel dan background terdekat bisa didapatkan dengan menggunakan fungsi distmap. 

nmask = watershed(distmap(nuc_th), 2)
display(colorLabels(nmask), all=TRUE)

Voronoi Tessellation

Vorronoi Tesselation sangat berguna untuk menggabungkan antara bibit objek satu dengan yang lainnya sehingga membentuk satu kesatuan. Fungsi ini ada didalam package (“EBImage”) dengan nama propogate. Kita akan mencoba menerapkannya dengan menggunakan nmask sebagai bibit dan menjadi partisi diantara mereka. 

voronoiExamp = propagate(seeds = nmask, x = nmask, lambda = 100)
voronoiPaint = colorLabels(voronoiExamp)
display(voronoiPaint)

Manipulasi Objek

Menghilangkan Objek

Package EBImage mendifinisikan setiap objek yang dimasking sebagai sebuah pixel yang memiliki nilai integer unik. Gambar yang memiliki objek masking merupakan hasil segmentasi dari fungsi seperti bwalabel, watershed, atau propagate. Untuk menghilangkan objek kita bisa menggunakan fungsi rmObject. Setelah objek terhapus kita bisa mengatur ulang ID objek dengan menggunakan reenumerate 

objects = list(
  seq.int(from = 2, to = max(logo_label), by = 2),
  seq.int(from = 2, to = max(logo_label), by = 2)
)
logos = combine(logo_label, logo_label)
z = rmObjects(logos, objects, reenumerate=FALSE)
display(z, all=TRUE)

Lalu kita coba untuk menampilkan ID dari gambar namun dengan reenumerate diset ke False. 

showIds = function(image) lapply(getFrames(image), function(frame) unique(as.vector(frame)))

showIds(z)

## [[2]]

Lalu kita panggil kembali ID objek yang ada dengan menggunakan standlone function reenumerate seperti dibawah ini 

showIds(reenumerate(z))

Filling Holes and Regions

Untuk mengisi lubang yang ada di objek kita bisa menggunakan fungsi fillHull. 

filled_logo = fillHull(logo)
display(filled_logo)

Lalu ada juga fungsi yang bernama floodFill yang berguna untuk mengisi daerah objek berdasarkan warna yang ditentukan. Tolerance merupakan perbedaan intensitas disetiap pixel. 

rgblogo = toRGB(logo)
points = rbind(c(50,50), c(100,50), c(150,50))
colors = c("red", "green", "blue")
rgblogo = floodFill(rgblogo, points, colors)
display(rgblogo)

Contoh Segmentasi Sel

Tujuan kita adalah untuk mengidentifikasi dan mengkualifikasi setiap karakter sel yang ada dengan komputasi. Kita coba untuk memproses gambar inti dan sel tubuh. Untuk menampilaknnya kita bisa meng-overlay gambar sampelnya dengan warna hijau dan biru seperti ini 

nuc = readImage(system.file('images', 'nuclei.tif', package='EBImage'))
cel = readImage(system.file('images', 'cells.tif', package='EBImage'))

cells = rgbImage(green=1.5*cel, blue=nuc)
display(cells, all=TRUE)

Pertama-tama, kita soba segmentasi inti ini menggunakan tresh, fillHull, bwlabel, dan opening. 

nmask = thresh(nuc, w=10, h=10, offset=0.05)
nmask = opening(nmask, makeBrush(5, shape='disc'))
nmask = fillHull(nmask)
nmask = bwlabel(nmask)

display(nmask, all=TRUE)

Lalu kita coba segmentasi lagi inti tadi menggunakan bibit yang ada didalam Vornoi Segmentation dari sitoplasma. 

ctmask = opening(cel>0.1, makeBrush(5, shape='disc'))
cmask = propagate(cel, seeds=nmask, mask=ctmask)

display(ctmask, all=TRUE)

Terakhir kita coba untuk memvisualisasikan segmentasi kita tadi menggunakan perintah paintObject. 

segmented = paintObjects(cmask, cells, col='#ff00ff')
segmented = paintObjects(nmask, segmented, col='#ffff00')

display(segmented, all=TRUE)
